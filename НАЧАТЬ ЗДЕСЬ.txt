🎯 ЧТО ЭТО?
════════════════════════════════════════════════════════════════

Программа автоматически создает документы:
• Акт приема-передачи товара
• Акт выполненных услуг

Из XLSX файлов с данными Wildberries.


🚀 КАК ЗАПУСТИТЬ?
════════════════════════════════════════════════════════════════

ШАГИ:

1️⃣  Установите Python 3.8+
    👉 https://www.python.org/downloads/
    ⚠️  При установке отметьте "Add Python to PATH"

2️⃣  Запустите файл двойным кликом:

    📂 macOS:
       "Запустить приложение.command"
       или
       "start.py"
       или
       "gui_app.py"  ← РЕКОМЕНДУЕТСЯ (чистое GUI)

    📂 Windows:
       "start.py"
       или
       "gui_app.py"  ← РЕКОМЕНДУЕТСЯ (чистое GUI)
       (если не открывается, кликните правой кнопкой
        → "Открыть с помощью" → Python)

3️⃣  Откроется окно программы - готово!
    ✨ НЕТ браузера, НЕТ localhost - только приложение!


📖 КАК ПОЛЬЗОВАТЬСЯ?
════════════════════════════════════════════════════════════════

1. Загрузите XLSX файл (перетащите или выберите)
2. Выберите нужный месяц из списка
3. Нажмите кнопку нужного документа
4. Документ скачается автоматически!


📁 ТРЕБОВАНИЯ К ФАЙЛУ
════════════════════════════════════════════════════════════════

✅ Формат: .xlsx (Excel)
✅ Каждая вкладка = месяц продаж
✅ Название вкладки = название месяца
✅ Первая строка = заголовки


❓ ПРОБЛЕМЫ?
════════════════════════════════════════════════════════════════

Проблема: Python не установлен
Решение: Скачайте с python.org

Проблема: Файл не запускается
Решение (macOS):
  1. Откройте Терминал
  2. Перетащите start.py в окно
  3. Нажмите Enter

Решение (Windows):
  1. Правой кнопкой на start.py
  2. "Открыть с помощью" → Python

Проблема: Ошибка при запуске
Решение: Откройте файл "README.md" для подробностей
