📌 САМЫЙ ПРОСТОЙ СПОСОБ (для новичков):

1. Установите Python с https://www.python.org/downloads/
   ⚠️ ВАЖНО: при установке отметьте галочку "Add Python to PATH"

2. Запустите двойным кликом один из этих файлов:

   macOS:
   ✅ "Запустить приложение.command"
   ✅ "start.py"

   Windows:
   ✅ "start.py"

3. При первом запуске программа сама установит все необходимое
   (это займет 1-2 минуты)

4. Откроется окно приложения - можно работать!

═══════════════════════════════════════════════════════════

🔧 ЕСЛИ НЕ ОТКРЫВАЕТСЯ:

Windows:
- Кликните правой кнопкой на "start.py"
- Выберите "Открыть с помощью" → Python

macOS:
- Откройте Терминал (Finder → Программы → Утилиты → Терминал)
- Перетащите файл "start.py" в окно Терминала
- Нажмите Enter
