# Changelog

Все важные изменения в проекте Wildberries Acts Generator.

## [2.1.0] - 2026-01-19

### Добавлено
- Столбец "Артикул продавца" в акт приёма-передачи
- Столбец "Артикул продавца" в акт оказания услуг
- Валидация артикулов продавца между двумя актами
- Предупреждение при обнаружении несоответствий в данных
- Скрипт deploy в package.json для быстрого деплоя
- CHANGELOG.md для отслеживания версий

### Изменено
- Обновлена структура таблиц в обоих актах
- Версия обновлена с 2.0.0 до 2.1.0

## [2.0.0] - 2026-01-19

### Добавлено
- Загрузка внешнего файла с брендами
- Поддержка колонки "Артикул продавца" в файле брендов
- Умное обнаружение заголовков в Excel файлах
- Приоритет данных из файла брендов над Excel файлом

### Изменено
- Улучшена логика извлечения брендов
- Оптимизирована обработка больших файлов

## [1.0.0] - 2026-01-19

### Добавлено
- Первый релиз статической React версии
- Генерация акта приёма-передачи
- Генерация акта оказания услуг
- Поддержка 119 специальных артикулов с брендами
- Экспорт через Google Docs
- Развёртывание на Vercel
