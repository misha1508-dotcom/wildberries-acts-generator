📋 СТРУКТУРА ФАЙЛА:

Ваш XLSX файл должен выглядеть так:

┌─────────────────────────────────────────────┐
│  Excel Файл: "Продажи WB 2024.xlsx"        │
├─────────────────────────────────────────────┤
│                                             │
│  📑 Вкладка: "Январь 2024"                  │
│  ┌────────────────────────────────────┐    │
│  │ Артикул│Товар    │Кол-во│Цена│Сумма│    │
│  │ 123456 │Платье   │  5   │1000│5000 │    │
│  │ 123457 │Брюки    │  3   │1500│4500 │    │
│  │ 123458 │Рубашка  │  10  │800 │8000 │    │
│  └────────────────────────────────────┘    │
│                                             │
│  📑 Вкладка: "Февраль 2024"                 │
│  ┌────────────────────────────────────┐    │
│  │ Артикул│Товар    │Кол-во│Цена│Сумма│    │
│  │ 123456 │Платье   │  7   │1000│7000 │    │
│  │ 123459 │Куртка   │  2   │3000│6000 │    │
│  └────────────────────────────────────┘    │
│                                             │
│  📑 Вкладка: "Март 2024"                    │
│  ...и так далее                             │
│                                             │
└─────────────────────────────────────────────┘


✅ ВАЖНЫЕ ПРАВИЛА:

1. Название вкладки = Название месяца
   • Январь 2024
   • Февраль 2024
   • Март 2024
   • и т.д.

2. Первая строка = Заголовки
   • Артикул, Название, Количество, Цена, Сумма
   • Можно использовать любые заголовки

3. Остальные строки = Данные
   • Каждая строка = один товар


📝 ПРИМЕРЫ НАЗВАНИЙ ВКЛАДОК:

Хорошо ✅               Плохо ❌
─────────────────────────────────────
Январь 2024            01
Февраль 2024           Sheet1
Март 2024              Лист1
Апрель 2024            tab1
Май 2024               !!!
Июнь 2024              (пустое)


🎯 ЧТО ПРОИСХОДИТ:

1. Вы загружаете файл в программу
2. Программа читает все вкладки
3. Каждая вкладка становится месяцем
4. Вы выбираете месяц
5. Программа генерирует 2 документа:
   • Акт приема-передачи (с таблицей всех товаров)
   • Акт услуг (с общими суммами)


💡 СОВЕТЫ:

• Используйте понятные названия вкладок
• Не оставляйте пустые строки в начале
• Заголовки должны быть в первой строке
• Можно добавлять любые колонки (комментарии, статус и т.д.)
